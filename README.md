# Smart Home Security System with Automatic Phone Calling using Arduino & IoT

## Overview
Designed and developed a smart home security system using Arduino and IoT technology to detect unauthorized motion and automatically alert users through phone calls or SMS notifications.

## Objective
To improve home security by providing real-time motion detection and instant user alerts using Arduino and wireless communication.

## Features
- Motion detection using PIR sensor
- Automatic phone call/SMS alert
- Real-time monitoring
- Arduino-based control system
- Cost-effective and scalable design

## Hardware Used
- Arduino Uno
- PIR Motion Sensor
- Bluetooth Module (HC-05)
- Power Supply
- Connecting Components

## Working Principle
The PIR sensor continuously monitors movement. When motion is detected:
1. Sensor sends signal to Arduino
2. Arduino processes the input
3. Bluetooth module triggers automatic call/SMS
4. User receives security alert

## Technical Specifications
- Operating Voltage: 5V DC
- Power Consumption: ~0.18 W
- PIR Detection Range: 3–7 meters
- Motion Detection Angle: ~120°

## Skills Applied
- Arduino Programming
- Sensor Interfacing
- IoT Basics
- Circuit Assembly
- Testing & Troubleshooting

## Outcome
Successfully developed a smart home security system capable of detecting motion and sending automatic alerts, improving safety and enabling quick response.
